PAM package¶

Module contents¶

The PAM package contains everything needed to run ProteinAnalysisManager.

–> Director module: Contains the Director class which orchestrates the processes contained in PAM.Builders and PAM.InputOutput

–> main module: Contains the UI_main(director) function, which runs PAM
