Representation module
=====================

The Representation module contains the Representation class, which have methods to coordinate user interface.
The user interface methods are in the same module.

********************
Representation Class
********************

The Representation class is the only class allowed to output anything, including printing and file out.
It also handles the user interface. Coordinating functions in the module, a Representation object handles all user input
and communicates with the Director class to change the state of data.
